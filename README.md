# BChess

> "I give 98 percent of my mental energy to BChess; others give only 2 percent"
> -- Bobby Fischer

BChess is a beginner-friendly chess in a console, with batteries included. 

# How to install

You can install the latest release version of BChess from [PyPI]
by running:

[PyPI]: https://pypi.org/project/bchess/

    python3 -m pip install --user bchess

Alternatively, you can install the development version by cloning
the repository:

    git clone https://github.com/magv/bchess
    cd bchess
    pip3 install --user .
